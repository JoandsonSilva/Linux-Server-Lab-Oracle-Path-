#Linux Server Lab (Oracle Path)
## Objetivo
Configurar um servidorlinux simulando ambiente de Data Center

---

## Etapas realizadas

-Instalacao do Ubuntu Server (arm64)
-Configuracao via UTM (Mac)
-Oriencatao de usuario
Ativacao de SSH

---

## Problema enfrentado

Sistema nao iniciava apos instalacao ( boot retornava para ISO)
---
\# Solucao

Acesso ao Boot Manager e selecao manual de opcao "Ubuntu"
---
##Comandos utilizados

-top
-free -m
who
is
pwd
